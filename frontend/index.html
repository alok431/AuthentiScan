How It Works

Seven layers of forensic analysis working together

01

Error Level Analysis

Detects JPEG compression inconsistencies by re-compressing and comparing error levels across the image.

02

Noise Analysis

Analyzes noise patterns across color channels. AI-generated images have distinctly different noise signatures.

03

Frequency Analysis

Examines spectral domain using FFT to detect GAN upsampling artifacts and checkerboard patterns.

04

Color Consistency

Analyzes color distribution, banding artifacts, and gradient smoothness to detect unnatural patterns.

05

Edge Detection

Uses Canny edge detection and Sobel filters to identify unnatural edge patterns common in deepfakes.

06

Texture Analysis

Local Binary Pattern analysis to detect GAN-specific repetitive texture patterns and low entropy regions.

07

Metadata Forensics

Examines EXIF data, camera information, GPS tags, and software signatures for signs of AI generation.
